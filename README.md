# 常用脚本
## 安装
### YUM-UPDATE
```
curl -o- https://raw.githubusercontent.com/G-YDG/scripts/master/linux/centos/yum-update.sh | bash
```
或者
```
wget -qO- https://raw.githubusercontent.com/G-YDG/scripts/master/linux/centos/yum-update.sh | bash
```
